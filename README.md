# IDEAS_Internship_TMT_Price_Prediction
Overview

Built a time-series forecasting solution to predict short-term steel price movements using historical and market-linked variables. The project compared classical and deep learning models to identify the most reliable forecasting approach for real-world decision support.

What I Worked On

Cleaned and prepared industrial time-series data
